## Methodology

### Data Sources

- **Wikimedia Pageviews API**: Daily pageview statistics with bot filtering
- **Wikipedia Edit History**: Revision data, editor metadata, vandalism signals
- **Crawl4AI**: Article content, infoboxes, categories, internal links

### Analytics Methods

**Time Series Forecasting:**
- Prophet model for seasonal data
- ARIMA for short-term predictions
- 180 days minimum training data
- 95% confidence intervals

**Reputation Monitoring:**
- Edit velocity calculation (edits/hour)
- Vandalism detection (revert analysis)
- Anonymous edit percentage
- Composite risk scoring (0-1 scale)

**Topic Clustering:**
- TF-IDF vectorization
- K-means clustering (k=20)
- Growth rate calculation
- CAGR computation

**Hype Detection:**
- View velocity analysis
- Edit growth tracking
- Content expansion measurement
- Composite hype scoring (0-1 scale)

### Validation

- **Statistical Significance**: p < 0.05 threshold
- **Confidence Intervals**: 95% CI for all predictions
- **Cross-Validation**: 30-day rolling window
- **Robustness Checks**: Multiple model comparison
